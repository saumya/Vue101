# VueJS Study App ( Vue 2 )

The basic commands are as below

```
npm install -g @vue/cli
vue create myApp
npm install vue-router
npm run serve
```


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
